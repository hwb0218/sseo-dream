# [써드림](https://sseo-dream-o5m6.vercel.app/)

## 개요
글쓰기 어려운 당신을 위해 대신 써드리는 서비스

## 프로젝트 소개
* 글쓰기에 어려움을 많이 느끼는 현대사회에서 글을 대신 써드리려는 생각에서 출발하였다.
* AI 기술을 가지고 작성 목적, 보내는 사람/받는 사람, 키워드, 길이만 입력하여, 그에 대한 글을 대신 쓰는 서비스를 만들었다.
